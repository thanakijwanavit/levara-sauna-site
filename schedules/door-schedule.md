## Door Types

### Type A: Sauna Glass Door (D1, D2, D3)
- **Glass:** Tempered, 8-10mm, clear or bronze tint
- **Frame:** Hinoki or treated wood, 60mm profile
- **Hinges:** 3× stainless steel, sauna-rated
- **Latch:** Magnetic catch only — no keyed locks
- **Handle:** Wood or silicone-wrapped (low thermal conductivity)
- **Seal:** Silicone gasket, heat-resistant

### Type B: Entry Door (D4)
- Per building code requirements
- May be standard interior or exterior rated
- Specification TBD based on corridor conditions

## Critical Notes

1. **All sauna doors swing outward** — this is a safety requirement for emergency egress
2. **No keyed locks on sauna doors** — magnetic latches only
3. **Minimum clear opening:** 700mm (800mm preferred)
4. **D3 swing direction** to be determined based on heater placement and bench layout

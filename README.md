# Pool-View Dual Sauna Suite — Hinoki

A luxury 3-room sauna suite featuring traditional electric and infrared saunas with pool views.

## Project Overview

| Dimension | Value |
|-----------|-------|
| Footprint | 2.95m (W) × 4.62m (L) |
| Ceiling Height | 3.50m (structural) |
| Total Capacity | 10 people (8 comfortable) |

## Rooms

1. **Vestibule / Cool-down** — Entry area with pool-facing window
2. **Infrared Sauna** — Lower temp (45–60°C), carbon/ceramic panels
3. **Electric Sauna** — Traditional high temp (75–90°C), stone heater

## Key Requirements

- Both saunas connected via internal door
- Dropped sauna ceilings at 2.2–2.3m for efficiency
- Pool-view window in vestibule (not saunas)
- Hinoki wood interior cladding
- Dedicated electrical circuits per heater type

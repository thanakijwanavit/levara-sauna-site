# Deliverables Checklist

Per design specs section 12, the following documents must be prepared for the contractor.

## Required Deliverables

### 1. Dimensioned Floor Plan

- [ ] Partition locations with thicknesses noted
- [ ] Bench layouts (L-shaped in each sauna)
- [ ] Door swings (all outward)
- [ ] Window placement and size
- [ ] Heater/IR panel locations
- [ ] Dimension strings (all critical measurements)

**Location:** `01-drawings/floor-plans/`

---

### 2. Interior Elevations

- [ ] Bench heights (seat: 450mm)
- [ ] Bench depths (500-550mm)
- [ ] Backrest angles (15-20° recline)
- [ ] Heater location and clearances
- [ ] IR panel layout
- [ ] Lighting positions
- [ ] Control panel location

**Location:** `01-drawings/elevations/`

---

### 3. Section Drawings

- [ ] Dropped sauna ceiling at 2.2-2.3m
- [ ] Full height vestibule (3.5m)
- [ ] Wall sandwich assembly detail
- [ ] Insulation and vapor barrier
- [ ] Service void above saunas
- [ ] Ventilation routes

**Location:** `01-drawings/sections/`

---

### 4. MEP Plan

- [ ] Electrical circuits (heater, IR, lighting)
- [ ] Panel locations and wire routes
- [ ] Ventilation supply/exhaust locations
- [ ] Duct routes
- [ ] Drain locations
- [ ] Plumbing rough-in (if shower included)

**Location:** `01-drawings/` (new file or combined)

---

### 5. Finish Schedule

- [ ] Hinoki grade specification
- [ ] Tile specification
- [ ] Glass/door hardware
- [ ] All finishes by room and surface

**Location:** `03-schedules/finish-schedule.md` ✓

---

## Acceptance Criteria (for verification)

From design specs section 13:

- [ ] Electric sauna achieves ~85°C reliably
- [ ] IR sauna reaches ~55°C with even coverage
- [ ] No persistent condensation damage around window
- [ ] Doors open smoothly under heat/humidity
- [ ] Ventilation prevents "stuffy" feeling at peak occupancy
